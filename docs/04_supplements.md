## 4-1. 하루 통합 타임라인 (전술 가이드 포함)

| 시간대 | 항목 (최종 수정본) | 전술 가이드 (비판적 반영) |
|--------|--------------------|-----------------------------|
| 기상 직후 | 유산균, 탈모약, SAT, 조인트 서포트 | 간 해독 & 관절 보호 시작 |
| 30분 후 | 올리브유(1스푼) + 레몬즙 | 칼로리 120kcal 절감. 지방 대사 스위치 |
| 아침 식사 | 케일주스, 두유, 합쎈 | 클린 탄수화물 및 식물성 단백질 |
| 점심 시간 | 업힐 16도 20분 * | 잉여 칼로리 연소 |
| 점심 식사 | 그릭요거트, 블루베리, 오메가3 | 항산화 및 클린 식단 |
| 낮 식후 | 루테인, 비타민D3+K2 | **이뮨샷 삭제.** 간 대사 부담 축소 및 필수 영양만 남김 |
| 오후 간식 | 유청 단백질(1스쿱), 달걀 1~2개 | 부족한 단백질 100g 구멍 메우기 |
| 저녁 식사 | 오메가3, 유청 단백질(1스쿱) | 근손실 방지 및 체지방 연소 환경 조성 |
| 운동 60분 전 | 콜라겐 + 레몬즙 | 관절/인대 합성 골든타임 |
| 운동 30분 전 | 크레아틴(5g), 타우린(2g), 베타알라닌(3g) | **시트룰린 삭제.** ATP 사전 장전으로 순간 폭발력(가속) 집중 |
| 운동 중 | 물 3.5L (전해질/소금) | 전해질 농도 사수 (반응 속도 유지) |
| 운동 직후 | 유청 단백질(1스쿱), 마그네슘 | 빠른 근육 회복 및 신경 이완 |
| 취침 전 | 테아닌(200mg), 타트체리 주스 | **이뮨샷 삭제.** 항염증 및 천연 멜라토닌으로 수면 효율 극대화 |

> \* **점심 세션은 요일별 플랜([02](02_training_plan.md))이 우선.** 업힐 16도 20분은 점심 헬스 세션이 없는 날(토 등)의 기본값. 금요일 점심은 어떤 운동도 X (파워냅).

## 4-2. 게임 데이 추가 (화/금/일)

| 타이밍 | 항목 |
|--------|------|
| 게임 30분 전 부스터 | 타우린 2g + 베타알라닌 3g (시트룰린 X, 카페인 X) |
| 게임 직후 | 유청 25g + 바나나 + 타르트체리주스 200ml |
| 게임 후 수면 스택 | 마그네슘 + 테아닌 + 글리신 3g |
| 야간 게임(금) 후 | + 멜라토닌 0.3mg |

## 4-3. 삭제/금지 목록 (다시 사기 전에 이 표부터)

| 항목 | 이유 |
|------|------|
| 이뮨샷 | 간 대사 부담. 필수 영양만 남김 |
| 시트룰린 | 지구력용 — 순간 폭발력(가속)에 불필요, 스택 단순화 |
| 카페인 부스터류 | 카페인 과민 + 13:00 컷오프와 충돌 |

## 4-4. 운영 룰

- 새 보충제 추가는 **한 번에 하나씩, 2주 간격** — 입병/피부/장 반응 원인 추적 가능해야 함
- 스택은 늘리는 게 아니라 지키는 것 — 위 표에 없는 건 기본 "안 삼"
- 물 3.5L는 운동일 기준. 전해질/소금 같이 (반응 속도 = 하키 퍼포먼스)
