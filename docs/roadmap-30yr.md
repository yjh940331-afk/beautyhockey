# 6. 30년 진화 로드맵

| 시기 | 빈도 | 강도 | 핵심 변화 |
|------|------|------|-----------|
| 현재 ~ 33세 | 3회 (조건부) | 동호인 페이스 | **무릎 통증 0 유지 사수** |
| 33~40세 | 2~3회 | 동호인 페이스 | 컨택트 대회 비중 ↓, 헤비 무게 ↓ |
| 41~50세 | 2회 | 친선 위주 | 회복 1주 가져갈 유연성, 플라이오 ↓ |
| 51~60세 | 1~2회 | 친선 only | 게임 ≤60분, 헤비 강도 ↓↓ |
| 61~70세 | 1회 | 사교적 | 시즌제 가능, 빙판이 친구 |

> **현재 3회는 라이프타임 빈도 피크 구간.** 평생 평균은 1.5~2회로 수렴함. 지금이 가장 많이 타는 시기.
