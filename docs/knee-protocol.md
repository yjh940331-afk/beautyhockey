# 2. 슬개건 보호 프로토콜 (The Knee Plan) ⚠️

**4회 시기 무릎 통증 = 30년의 1순위 변수.**
주간 스케줄에 이미 통합돼 있지만, 따로 정리.

---

## 주간 슬개건 자극 분포 (주 5회, 매번 단시간)

| 언제 | 무엇 |
|------|------|
| 월 저녁 | 스패니시 ISO 45초 × 5 |
| 수 점심 | 데클라인 편심 8×3 + 스패니시 ISO 45초 × 3 (헤비 하체와 통합) |
| 목 저녁 | 리버스 노르딕 5×3 (편심) |
| 토 저녁 | 스패니시 ISO 45초 × 5 |
| 게임 직전 (화/금/일) | 스패니시 ISO 45초 × 5 ← 진통 효과 |

---

## 증량 룰

- **데클라인 스쿼트 편심:** 8회 다 통증 없이 끝나면 +2.5kg
- **스패니시 스쿼트:** 밴드 강도 ↑ 또는 거리 ↑

---

## 통증 트래픽 라이트

| 신호 | 행동 |
|------|------|
| 🟢 0~3/10 | 정상 진행 |
| 🟡 4/10 | 그 세트 즉시 중단, 다음 세션 무게 50% 감량 |
| 🔴 5+/10 또는 4일 이상 지속 | **즉시 1주 하키 휴식 (협상 X)**, 통증 0/10 복귀 시까지 헬스도 무릎 자극 0 |
| 🚨 슬개골 자체에 콕 찌르는 통증 | **즉시 정형외과** (스트레스 반응 배제) |
