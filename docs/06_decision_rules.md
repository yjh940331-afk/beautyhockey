# 06. 결정 룰 — 세 목표가 충돌할 때 이 문서가 판정한다

**우선순위 고정: ① 무릎/건강 → ② 하키 퍼포먼스 → ③ 컷팅/외모.**
아래 룰과 감정/의지가 충돌하면 룰이 이긴다.

---

## 6-1. 5대 원칙 (헌법)

1. **무릎이 트레이닝의 상한선.** 통증 4일 이상 = 즉시 1주 휴식 (협상 X)
2. **수면 = 회복의 단일 최강 변수.** 수면 80% 미만 주 2연속 → 다음주 강제 디로딩
3. **헤비 자극은 게임 72시간 전에 끝낸다.** 하체 헤비 = 수요일만, 플라이오 = 목요일만
4. **카페인은 13:00까지** (금요일 12:00)
5. **하루 3세션 = 목요일만.** 다른 요일 최대 2세션

## 6-2. 슬개건 프로토콜 (The Knee Plan)

주간 자극 분포 (주 5회, 매번 단시간):

| 언제 | 무엇 |
|------|------|
| 월 저녁 | 스패니시 ISO 45초 × 5 |
| 수 점심 | 데클라인 편심 8×3 + 스패니시 ISO 45초 × 3 |
| 목 저녁 | 리버스 노르딕 5×3 (편심) |
| 토 저녁 | 스패니시 ISO 45초 × 5 |
| 게임 직전 (화/금/일) | 스패니시 ISO 45초 × 5 ← 진통 효과 |

증량 룰: 데클라인 편심 8회 무통 완수 → +2.5kg / 스패니시 → 밴드 강도↑ 또는 거리↑

| 통증 신호 | 행동 |
|------|------|
| 🟢 0~3/10 | 정상 진행 |
| 🟡 4/10 | 그 세트 즉시 중단, 다음 세션 무게 50% |
| 🔴 5+/10 또는 4일 지속 | **즉시 1주 하키 휴식**, 통증 0/10까지 무릎 자극 0 |
| 🚨 슬개골에 콕 찌르는 통증 | **즉시 정형외과** (스트레스 반응 배제) |

## 6-3. 전신 트래픽 라이트

| 신호 | 행동 |
|------|------|
| RHR +5bpm / 1일 | 그날 강도 -20% |
| RHR +5bpm / 3일 연속 | 즉시 디로딩 1주 |
| RHR +8bpm | 빨간불 — 그날 완전 회복만 |
| 동기 5↓ + RHR↑ 2일 | 완전 휴식 |
| 게임 중 평소보다 빨리 방전 | 다음주 점심 헬스 1회 제거 |
| 수면 80% 미달 주 2연속 | 다음주 강제 디로딩 |

## 6-4. 목표 충돌 판정표

| 충돌 상황 | 판정 |
|-----------|------|
| 컷팅 중인데 게임력 하락 | 컷팅 중단 (②>③). 유지 칼로리 복귀, 2주 후 재시도 |
| 컷팅 중인데 무릎 통증 ↑ | 적자 즉시 0 (①>③). 저칼로리 = 건 회복력 ↓ |
| 게임 뛰고 싶은데 무릎 🔴 | 휴식 (①>②). 협상 없음 |
| 피부 트러블 vs 훈련 | 훈련 유지, 원인은 장비 세척/세안/식단에서 찾기 |
| 입병 발생 | 수면 +30분 우선, 그 주 적자 0, 강도 유지 여부는 컨디션으로 |
| 회식/술자리 vs 플랜 | 술 0 고정 ([03](03_nutrition_cutting.md) 3-4), 참석은 자유 |
| 디로딩 주에 컨디션 좋음 | 그래도 디로딩 (좋게 느껴지는 게 디로딩이 작동한다는 뜻) |

## 6-5. 체질 고정 룰 요약

| 조건 | 룰 |
|------|-----|
| 알코올 | 기본 0잔, 마신 날 기록 + 다음날 강도 -20% |
| 카페인 과민 | 13:00 컷오프, 주 1회 0 데이 (일) |
| 햇빛알러지 | 무기자차 SPF50+ 매일, 장시간 야외 재도포 ([05](05_skin_appearance.md)) |

| 입병 | SPF 립밤, 재발 잦으면 피부과 확진 |
