# 05. 외모 · 피부 · 선크림

"가장 잘생긴"의 실체 = **체지방 (윤곽) + 자세 (거북목 교정) + 피부 (트러블 0) + 모발 (탈모약 유지)**.
앞 두 개는 01/02 문서가 담당. 이 문서는 피부/햇빛/입병.

---

## 5-1. 햇빛알러지 대응 (선크림 기준)

- **무기자차 (미네랄 선크림: 징크옥사이드/티타늄디옥사이드), SPF50+ / PA++++** — 유기자차는 햇빛알러지 피부에 자극 가능성 ↑
- 매일 아침 마지막 스텝으로 도포. **흐린 날도 예외 없음**
- 2시간 이상 야외 (토요일 16:00 산책 등): 외출 직전 재도포 + 모자
- 아침 "햇빛 5분" (생체시계용)은 유지 — 짧아서 OK, 단 얼굴은 선크림 후
- 새 선크림은 팔 안쪽 2일 패치 테스트 후 얼굴에
- **현재 사용 제품: __________ (기입할 것)**

## 5-2. 입병 관리 (HPV/헤르페스 의심)

- **트리거 3대장: 수면 부족 · 게임 스트레스 · 햇빛** → 이 플랜의 수면/디로딩 룰이 곧 입병 예방
- **립밤도 SPF 있는 걸로** — 입술 자외선 노출은 헤르페스 재발의 검증된 트리거
- 발생 시 daily log에 날짜 기록 → 직전 3일 수면/게임/햇빛 패턴 확인
- 재발이 잦으면 (월 1회 이상) 피부과 진료 — 항바이러스제 처방 상담. 의심 상태로 방치하지 말고 **한 번은 확진 받기**
- 발생 중엔 수건/립밤 공유 금지, 터뜨리지 않기

## 5-3. 피부 기본 루틴

| 때 | 루틴 |
|----|------|
| 아침 | 미온수 세안 (약산성 클렌저) → 보습 → **선크림** |
| 게임/운동 직후 | 땀 바로 세안 (헬멧/장비 접촉 부위 트러블 방지) — 찬물 샤워 프로토콜과 겸함 |
| 밤 | 클렌징 → 보습. 자극 성분 새로 추가할 땐 한 번에 하나 |
| 주 1회 | 헬멧 내피/친가드 세척 (피부 접촉 장비 = 트러블 원인 1순위) |

- 히스타민 과민 = 피부 반응 잘 나오는 체질 → **새 화장품도 보충제처럼 하나씩 2주 간격**
- 현재 사용 제품 리스트: __________ (기입할 것)

## 5-4. 외모 측정 (주 1회, 수요일)

- 거울 사진 정면+측면 (같은 조명/장소/시간) — 컷팅 진행의 진짜 지표
- 자세 사진 측면 (귀-어깨 정렬) — 거북목 교정 진행 확인, 월 1회 비교
