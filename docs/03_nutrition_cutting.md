# 03. 영양 + 컷팅

체질 전제: **히스타민 과민증 · 장과민증 · 알코올중독 초기증상 · 카페인 과민증** → 룰은 [06_decision_rules.md](06_decision_rules.md)에도 반영.

## 3-1. 단백질 분배 (목표 160g, 80kg × 2g)

3-2 표준 메뉴 기준 추정치:

| 시간 | 식단 | 단백질 |
|------|------|--------|
| 아침 | 오트밀 + 계란 흰자 3~4개 + 두유 | ~28g |
| 점심 | 현미밥 + 소고기 우둔살 200g | ~42g |
| 간식 | 꾸덕이 요거트 220g + 견과 (또는 유청+달걀) | ~22g |
| 저녁 | 흰살생선/닭안심 + 유청 1스쿱 | ~55g |
| 운동 직후 (운동일) | 유청 25g | ~25g |
| **합계** | | **비운동일 ~147g / 운동일 ~172g** |

> 비운동일(사실상 월요일 정도)에 부족하면 간식에 달걀 1~2개 추가로 160g 맞춤. 세 자리 정확도로 집착하지 말 것 — **매 끼니 단백질 주인공 하나**가 룰의 실체.

## 3-2. 표준 하루 메뉴 구성

| 구분 | 메뉴 구성 | 기대 효과 |
|------|-----------|-----------|
| 아침 | 오트밀 + 블루베리 + 계란 흰자 3~4개 + 토마토주스 + 두유 | 지속적인 에너지 공급 및 근육 유지 |
| 점심 | 현미밥 1공기 + 소고기 우둔살 (200g) + 구운 브로콜리 | 철분 보충 및 파워 유지 |
| 간식 | 요거트 + 견과류 한 줌 (또는 프로틴 바) | 허기 방지 및 아미노산 공급 |
| 저녁 | 흰살생선 또는 닭안심 + 단호박 + 신선한 샐러드 | 소화 부담 최소화 및 항염 작용 |

- 간식 상세: **꾸덕이 요거트(220g) + 블루베리 + 호두 + 아몬드(소량) + (키위 추가)**
- 아침 시작: **올리브유 + 레몬즙** (기상 30분 후, [04](04_supplements.md) 타임라인과 동일)

> 운용: 이 표가 기본 틀. 3-1 단백질 분배표는 총량(160g) 체크용, 3-3 칼로리 사이클링은 요일별 양 조절용 — 월/토는 이 메뉴에서 탄수(현미밥/오트밀)를 줄여 -400을 만든다. 게임일 저녁은 소화 부담 최소 구성(흰살생선/닭안심)이 특히 중요.

## 3-3. 칼로리 사이클링

| 요일 | 칼로리 | 탄수 비중 | 이유 |
|------|--------|-----------|------|
| 월 | -400 (적자) | 30% | 회복일, 활동 적음 |
| 화 | 유지 | 50% | 게임 |
| 수 | 유지 | 45% | 헤비 하체 |
| 목 | 유지~+200 | 50% | 풀파워 데이 |
| 금 | 유지 | 50% | 게임 |
| 토 | -400 (적자) | 35% | 회복+적자 |
| 일 | 유지 | 50% | 게임 |
| **주 평균** | **-230/일** | | 6주 ≈ 1.4kg 지방 감량 |

> 12% 이하 구간의 추가 룰은 [01_body_composition_goal.md](01_body_composition_goal.md) 참조.

## 3-4. 카페인 룰 (절대)

- 7:00~13:00 사이만 OK
- **13:00 이후 0** (디카페인도 의심됨)
- **금요일은 12:00 컷오프** (야간 게임 대비)
- 주 1회 카페인 0 데이 → **일요일 추천**

## 3-5. 알코올 룰 (절대)

- 알코올중독 초기증상 = **기본값 0잔.** "게임 후 한 잔"도 없음 — 회복(수면·단백질 합성·슬개건)을 직접 깎는다
- 사교 자리: 무알콜/탄산수로 대체, 술자리 자체를 게임 전날엔 회피
- 마신 날은 daily log에 반드시 기록 (숨기면 시스템이 죽는다) → 다음날 강도 -20% + 적자 금지

## 3-6. 히스타민/장 과민 대응

- ⚠️ **표준 메뉴 안에 히스타민 의심 항목이 있다: 토마토주스(대표적 히스타민 유리 식품), 키위, 호두.** 금지는 아니고 — 2주간 daily log에 피부/입병/장 반응을 같이 기록해서 패턴 보고 판단. 반응 있으면 토마토주스 → 다른 채소주스/과일로 교체
- 히스타민 주의: 숙성육, 훈제/가공육, 오래된 생선, 발효식품 과다, 남은 음식 재가열 반복
- 게임 전 식사(18:30)는 **신선 조리 + 단순 구성** (현미+닭가슴살이 표준인 이유)
- 그릭요거트/발효식품에 피부·장 반응 오면 daily log에 기록 → 2주 패턴 보고 교체 판단
- 장 트러블 난 날: 게임 전 식사량 절반 + 소화 잘 되는 탄수(흰쌀/바나나)로 전환 OK
