| 요일 | 칼로리 | 탄수 비중 | 이유 |

## 3-3. 카페인 룰 (절대)

- 7:00~13:00 사이만 OK
- **13:00 이후 0** (디카페인도 의심됨)
- **금요일은 12:00 컷오프** (야간 게임 대비)
- 주 1회 카페인 0 데이 → **일요일 추천**

## 3-4. 알코올 룰 (절대)

- 알코올중독 초기증상 = **기본값 0잔.** "게임 후 한 잔"도 없음 — 회복(수면·단백질 합성·슬개건)을 직접 깎는다
- 사교 자리: 무알콜/탄산수로 대체, 술자리 자체를 게임 전날엔 회피
- 마신 날은 daily log에 반드시 기록 (숨기면 시스템이 죽는다) → 다음날 강도 -20% + 적자 금지

## 3-5. 히스타민/장 과민 대응

- 히스타민 주의: 숙성육, 훈제/가공육, 오래된 생선, 발효식품 과다, 남은 음식 재가열 반복
- 게임 전 식사(18:30)는 **신선 조리 + 단순 구성** (현미+닭가슴살이 표준인 이유)
- 그릭요거트/발효식품에 피부·장 반응 오면 daily log에 기록 → 2주 패턴 보고 교체 판단
- 장 트러블 난 날: 게임 전 식사량 절반 + 소화 잘 되는 탄수(흰쌀/바나나)로 전환 OK
