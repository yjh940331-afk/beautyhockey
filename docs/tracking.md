# 5. 자가 측정 + 중단 신호

측정 → 조정 → 측정. 이 사이클이 진짜 시스템이다. 기록은 [log/](../log/)에.

---

## 5-1. 매일 아침 (30초)

- **안정시 심박수(RHR)** 기상 직후: 평균에서 +5bpm 이상 = 노란불, +8 이상 = 빨간불
- **수면** 시간 + 1~10점 자가평가
- **무릎 통증** 0~10점

## 5-2. 매주 수요일 아침 (5분)

- 체중 (참고용)
- 허리둘레 (배꼽 위 2cm 줄자)
- 거울 사진 (정면+측면, 같은 조명)
- 동기 1~10점

---

## 5-3. 트래픽 라이트 (행동 룰)

| 신호 | 행동 |
|------|------|
| RHR +5 / 1일 | 그날 강도 -20% |
| RHR +5 / 3일 연속 | 즉시 디로딩 1주 시작 |
| 동기 5↓ + RHR↑ 2일 | 완전 휴식 |
| 무릎 통증 4+/10 | 그날 모든 무릎 자극 X |
| 무릎 통증 4일 지속 | **1주 하키 휴식 (필수)** |
| 게임 중 평소보다 빨리 지침 | 다음주 점심 헬스 1회 빼기 |
| 슬개골 자체 콕 찌르는 통증 | **즉시 정형외과** |

---

## 5-4. 6주 마일스톤

- [ ] 체중 80kg, 체지방 12%
- [ ] 허리둘레 -2cm
- [ ] 거울 복근 윤곽 ↑
- [ ] 하키 첫 3스트라이드 가속 측정 (영상 비교)
- [ ] 양발 스탑 출력 균형 점검
- [ ] **무릎 통증 0/10 평균 ← 이게 가장 중요**

6주 데이터가 쌓이면 그걸로 플랜 재조정.
